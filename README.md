# Sales Analysis Dashboard

Interactive Power BI dashboard analysing 1,000 orders 
with Total Sales ₹16,91,52,612 and Profit ₹2,76,05,209 
across 4 regions and 5 cities.

## Visuals
- Sales by Month, Category, City, Region
- Salesperson Performance
- Payment Mode & Order Status breakdown

## Tools Used
Power BI | DAX | Advanced Excel
